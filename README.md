# Formless

> Replace your forms with a conversation.

Formless is an AI-powered intake platform that lets employees submit requests, tickets, and
structured data in plain English — no forms, no fields, no training required.

**The idea:** Forms exist because computers can't understand intent. LLMs changed that.

---

## How It Works

1. **Admin defines a schema** — in plain English: *"I need IT tickets with a title, description, category, and priority"*
2. **Formless generates the data structure** — typed fields, required flags, enum values
3. **End users submit in plain English** — *"My laptop won't connect to VPN and I have a client call in an hour"*
4. **Formless classifies, extracts, and validates** — structured JSON record, ready to route
5. **Record goes where it needs to go** — webhook, Jira, Slack, email

---

## Use Cases

- IT helpdesk tickets
- HR onboarding intake
- Procurement requests
- Incident reports
- Legal intake
- Customer support triage
